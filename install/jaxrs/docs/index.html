JAX-RS 2.1 TCK Documentation

The JAX-RS 2.1 TCK documentation includes the following:

  • The JAX-RS 2.1 Release Notes, provide late-breaking information about the JAX-RS 2.1 TCK

  • The JAX-RS 2.1 Technology Compatibility Kit User's Guide provides the information that you need to install, set up, and run the JAX-RS 2.1 TCK. The guide also provides the rules with which you must comply to pass the JAX-RS 2.1 TCK.

  • The Javadoc Assertion List includes the javadoc assertions that are indirectly tested by the JAX-RS 2.1 TCK.

  • The Specification Assertion List includes the specification assertions that are tested by the JAX-RS 2.1 TCK.
